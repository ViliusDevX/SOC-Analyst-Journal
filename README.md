# SOC Analyst Journal

Live Demo: https://soc-journal-kappa.vercel.app/

SOC Analyst Journal is a long term cybersecurity learning project focused on SOC analysis, CySA+ preparation and incident investigation.

The project combines:
- SOC investigation scenarios
- CySA+ practice questions
- Alert triage and incident response reasoning
- MITRE ATT&CK oriented thinking
- Interactive React based practice systems
- Public learning progression over time

---

## Current Features

- 90 CySA+ / SOC practice questions
- 15 investigation scenarios
- Structured JSON-based questionnaire system
- Scenario review/checking system
- Interactive Scenario Lab
- React + Tailwind frontend
- Learning progression tracking

---

## Purpose

I created this project to do three main things:

### 1. Capture My Learning Progression
Documenting raw answers, mistakes, improvements, and reasoning over time.

### 2. Showcase My Skills
Combining cybersecurity learning with frontend development using React, Tailwind CSS, reusable components, routing, and structured JSON data.

### 3. Help Others Practice SOC Skills for Free
The long-term goal is to build a free SOC practice platform where users can learn investigation thinking through realistic scenarios and guided explanations.
